# 🐾 Pet Adoption Platform (Server)

Backend API of the Pet Adoption Platform built with Express.js, MongoDB, and JWT Authentication.

---

# 🌐 Server URL

https://server-two-xi-91.vercel.app/

---

# 📌 Purpose

This backend server handles:

- User Authentication
- Pet Management
- Adoption Requests
- Protected Routes
- Secure Cookie Handling
- Database Operations

---

# 🚀 Features

- 🔐 JWT Authentication
- 🍪 HTTPOnly Cookie Support
- 🔒 Protected API Routes
- 🐶 Add / Update / Delete Pets
- ❤️ Adoption Request System
- 📩 Approve & Reject Requests
- 🚫 Prevent Self Adoption
- ✅ One-Time Pet Adoption Logic
- 🌐 CORS Configured
- ⚡ Error Handling Middleware

---

# 🛠️ Technologies Used

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcrypt
- cookie-parser
- cors
- dotenv
